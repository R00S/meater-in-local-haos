## Rules

1. Each leaf file must use 4–6 sources from DIFFERENT traditions.
2. Local/non-English sources are searched FIRST.
3. Trusted English fallbacks (Chefkoch, BBC Good Food, RecipeTin Eats) are used
   only when local sources cannot fill the required diversity.
4. No source may be used twice within the same leaf file.
5. AI-generated content is strictly prohibited as a source.

## Internet Archive Methodology (discovered 2026-04-24)

The sandbox cannot reach recipe sites directly (DNS blocked). **web.archive.org IS
reachable** and must be used to fetch all recipe sources.

### Correct workflow

1. Find snapshot timestamp via CDX API (wildcard `*` required):
   ```
   https://web.archive.org/cdx/search/cdx?url=DOMAIN/PATH*&output=text&fl=timestamp,original&limit=1
   ```
2. Fetch archived page using the **full 14-digit timestamp** from CDX output:
   ```
   https://web.archive.org/web/YYYYMMDDHHMMSS/https://ORIGINAL_URL
   ```

### Critical rules
- Wildcard suffix `*` is **required** in CDX queries — bare URL returns TypeError.
- Full 14-digit timestamp is **required** in page fetches — 8-digit date only returns TypeError.
- Use `web_fetch` rather than `curl` — more reliable for archived pages.
